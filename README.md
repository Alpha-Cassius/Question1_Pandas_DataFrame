
# Pandas Data Exploration Assignment

This repository contains the solution to the **Pandas DataFrame** exploration exercise. The notebook demonstrates how to create, save, reload, and perform basic statistical analysis on student data.

## Contents

* `Question1_Pandas_DataFrame.ipynb`: The main Jupyter notebook containing the code and outputs for the assignment.
* `students.csv`: The dataset generated during the execution of the notebook.

## Getting Started

To view this project:

1. **Directly on GitHub:** Simply click on the `.ipynb` file in this repository. GitHub will render the code and the outputs automatically.
2. **In Google Colab:** Click the **"Open in Colab"** button (if you added the badge) or download the `.ipynb` file and upload it to [colab.research.google.com](https://colab.research.google.com).

## Implementation Details

The notebook covers the following tasks:

* Creating a DataFrame from a dictionary.
* Exporting and importing data using CSV files.
* Inspecting data with `head()`, `describe()`, `shape`, `dtypes`, and `info()`.
